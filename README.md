# N225 Watcher (日経平均株価モニター)

日経平均株価（N225）の最新価格、前日比、騰落率、および期間別の株価推移チャートを美しく表示するWebアプリケーションです。

## 主な機能
- **自動更新**: 起動時および1分ごとに自動で日経平均株価を更新（手動更新ボタンも搭載）。
- **株価推移チャート**: 7日、1ヶ月、3ヶ月、1年間のヒストリカルチャートを折れ線グラフ（エリアグラデーション付き）で表示。
- **プレミアムなUI**: ガラスモーフィズム（Glassmorphism）と深みのあるダークテーマを採用したモダンなダッシュボードデザイン。前日比がプラスかマイナスかに応じて、UIの配色テーマ（エメラルドグリーン / ローズレッド）が動的に変化します。
- **完全サーバーレス対応**: CORSプロキシを経由して Yahoo Finance からデータを直接フェッチするため、GitHub Pages などの静的ホスティング環境でそのまま動作します。

## 技術スタック
- フロントエンド: HTML5 / Vanilla CSS / JavaScript (ES6)
- グラフ描画: Chart.js (CDN)
- アイコン: Lucide Icons (CDN)
